# Geometry Art Set

This folder contains simple geometric placeholder sprites for the top-down roguelite prototype.

## Characters

- `Characters/player_core.png`: main player marker.
- `Characters/player_dash_shadow.png`: movement or dash afterimage.

## Enemies

- `Enemies/enemy_triangle_chaser.png`: fast chasing enemy.
- `Enemies/enemy_square_guard.png`: sturdy guard enemy.
- `Enemies/enemy_diamond_rusher.png`: direct rushing enemy.
- `Enemies/enemy_circle_spitter.png`: ranged/spitting enemy.

## Weapons

- `Weapons/weapon_knife.png`
- `Weapons/weapon_sword.png`
- `Weapons/weapon_spear.png`
- `Weapons/weapon_spell_orb.png`

## Tiles

- `Tiles/tile_ground_grid.png`: generic combat floor.
- `Tiles/tile_safe_room_floor.png`: safe-room floor.
- `Tiles/tile_danger_floor.png`: hostile stage floor.
- `Tiles/tile_wall_block.png`: wall/block tile.
- `Tiles/tile_safe_door.png`: safe-room entrance/exit marker.

## VFX And UI

- `VFX/attack_hit_circle.png`: melee hit marker.
- `VFX/attack_arc_wedge.png`: attack direction wedge.
- `VFX/danger_telegraph.png`: enemy warning area.
- `Pickups/pickup_energy_shard.png`: resource pickup.
- `Pickups/pickup_coin_hex.png`: currency pickup.
- `UI/icon_safe_room.png`: map safe-room icon.
- `UI/icon_stage_node.png`: map stage-node icon.

`Player.png` and `Ground.png` are root-level aliases kept for older scene references.
